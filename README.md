Relógio Digital Neon com Tema Claro/Escuro

Projeto de um relógio digital moderno e estilizado que mostra a hora e a data atual, atualizado a cada segundo. Possui efeito neon e botão para alternar entre modo claro e modo escuro, proporcionando um visual dinâmico e futurista.

Funcionalidades

Atualização em tempo real da hora (HH:MM:SS) e data em português brasileiro.

Botão para alternar entre tema claro e escuro com efeitos neon.

Layout responsivo, adequado para desktops e dispositivos móveis.

Visual moderno inspirado em estilo cyberpunk/neon com sombras e gradientes.

Tecnologias Utilizadas

HTML5 – Estrutura do relógio.

CSS3 – Estilo neon, animações e temas claros/escuros.

JavaScript – Atualização do relógio e alternância de temas.

Estrutura do Projeto
/assets
  ├─ style.css       # Estilos do relógio e efeitos neon
  └─ script.js       # Lógica do relógio e troca de tema
index.html            # Página principal com o relógio
README.md             # Documentação do projeto

Uso

Abra o arquivo index.html em qualquer navegador moderno.

O relógio e a data serão atualizados automaticamente a cada segundo.

Clique no botão Mudar Cor para alternar entre modo claro e modo escuro.
